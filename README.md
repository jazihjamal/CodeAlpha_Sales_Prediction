# Sales Prediction Using Python

## 📌 Project Overview
This project predicts product sales based on advertising budgets using Machine Learning.

The model analyzes advertising expenditure on:
- TV
- Radio
- Newspaper

to predict product sales.

---

## 🛠️ Technologies Used
- Python
- Google Colab
- Pandas
- Scikit-learn
- Matplotlib

---

## 🤖 Machine Learning Algorithm
- Linear Regression

---

## 📂 Dataset
Advertising dataset containing TV, Radio, Newspaper advertising budgets and Sales.

---

## 📊 Results
- R² Score: **0.8994**
- Mean Absolute Error: **1.4608**

---

## 🎯 Learning Outcomes
- Data preprocessing
- Linear Regression
- Model training and evaluation
- Data visualization
